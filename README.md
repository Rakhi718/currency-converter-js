# Currency Converter

A currency converter web application that converts one currency to another using live exchange rates.

## Features
- Real-time currency conversion
- Multiple currencies supported
- Dynamic exchange rate fetching
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Currency API

## How to Run
1. Download the project.
2. Open `index.html` in your browser.
